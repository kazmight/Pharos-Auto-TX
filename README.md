## Installation

```bash
git clone https://github.com/kazmight/Pharos-Auto-TX.git
cd Pharos-Auto-TX
```

**Install Dependencies**:
```bash
npm install
```

To run the bot, use the following command in your terminal:

```bash
node index.js
```
